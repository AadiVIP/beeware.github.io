O Bee Team

O time central por trás do desenvolvimento do BeeWare

Quem são as pessoas por trás do BeeWare? Existe um enorme grupo de contribuidores, mas o projeto é administrado pelo Bee Team.

Russell Keith-Magee

Dr Russell Keith-Magee é um veterano com 10 anos no time principal do Django, e por 5 anos foi presidente da Django Software Foundation. Ele também é o fundador do projeto BeeWare, desenvolvendo GUI tools para dar suporte ao desenvolvimento de software Python. Quando ele não está contribuindo para open source, ele é o CTO da TradesCloud, uma companhia que provê software de gerenciamento integrado de trabalho para comerciantes.

Katie McLaughlin

Katie ganhou muitos chapéis diferentes ao longo dos anos. Ela já desenvolveu software para muitas linguagens, administradora de sistema para múltiplos sistemas operacionais e palestrante de diversos assuntos.

Quando ela não esta mudando o mundo, gosta de cozinhar, fazer tapeçarias e gritar com o JavaScript e sua tentativa de variáveis globais.

Philip James

Philip escreve código para computadores e para humanos. Seu primeiro amor computacional é Python, mas as vezes é forçado a escrever JavaScript e Swift. Quando ele não está codando por dinheiro ou para Open Source para seus projetos loucos, ele palestra em conferências. Philip vive em Alameda, California com sua esposa e seu gato. Philip acredita na web.

Christopher Swenson

Dr. Christopher Swenson é um engenheiro de software, cientista da computação e ocasionalmente um matemático. Swenson trabalha com Python desde 2004, e envolvido com open source desde antes de saber o que open source significava. Swenson gosta muito de ordenação, e mantém uma biblioteca open source de alta performance em C para ordenação. Swenson ja vendeu seu trabalho para companhias grandes e pequenas, como Simple, Google e o governo dos EUA e escreveu um livro de criptografia. Swenson é um organizador da PyDX. Swenson é um cupcake.

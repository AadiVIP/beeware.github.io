In addition to support for native widgets, Toga can support a web browser as an output platform.

When deployed to the browser, Toga uses Batavia to run your application's code. Batavia is an implementation of a CPython bytecode virtual machine, written in Javascript.

Follow the tutorial in the briefcase docs to get started!
